Project Title:IITshuttlebus

Team Members:Madhumitha,Amanda,Ernest,Phoenix.


Language used: java mob edition

IDE:Android studio

Also can be run as Android Project in Eclipse.


Steps:

1.unzip the folder iitShuttleBus.

2.open Android  studio

3.Click on file import project

4.Goto the location of the unzipped folder iitShuttleBus and select it

5.Select the gradle repository

6.Click next and finish



To run:

1.Under the project iitShuttleBus,right click on  the src folder
 and run as android application
